La tarta is a clean and modern theme for your online Blog website. It is a perfect choice for personal blog, corporate blog or any type of creative blog. The design is very clean, elegant and modern.

The theme pack includes the detail pages too. It makes the theme easy to develop for blogging framework like Wordpress.

The layout is developed on the bootstrap grid, which makes it easy to convert it into an adaptive html template, WordPress, Joomla, WooCommerce and other CMS.
